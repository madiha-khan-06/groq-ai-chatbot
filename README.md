## 📖 Overview

**Groq AI Assistant** is a modern AI chatbot designed to deliver **real-time, ultra-fast responses** using Groq’s powerful LLM infrastructure.

### 🧩 Built With:
- ⚡ **Groq LLM (LLaMA 3.3)**
- 🔗 **LangChain Framework**
- 🎨 **Streamlit Interface**

### 💭 What You Can Do:
- 💬 Ask questions naturally  
- 🧠 Receive AI-powered responses  
- 🎚️ Control model behavior dynamically  

---

## 🌟 Features

- 💬 Interactive chat experience  
- ⚡ Instant AI response generation  
- 🎚️ Adjustable temperature control  
- 🔐 Secure API key integration  
- 🧩 Clean & beginner-friendly code  
- 📱 Minimal and responsive UI  

---

## ⚙️ How It Works

1. User enters a query 💬  
2. LangChain processes input 🔗  
3. Groq LLM generates output ⚡  
4. Streamlit renders response 🎨

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/groq-ai-ChatBot.git
cd groq-ai-assistant
````

---

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

### 3️⃣ Activate Environment

#### Windows:

```bash
venv\Scripts\activate
```

#### Mac/Linux:

```bash
source venv/bin/activate
```

---

### 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Add API Key

Get your API key from 👉 [https://console.groq.com/](https://console.groq.com/)

Then either:

### Option 1: Streamlit Sidebar (recommended)

Paste directly in app UI

### Option 2: Environment Variable

```bash
export GROQ_API_KEY="your_key"
```

---

## ▶️ Run the App

```bash
streamlit run app.py
```

---

## 🌐 Deployment (Streamlit Cloud)

1. Push code to GitHub
2. Go to [https://streamlit.io/cloud](https://streamlit.io/cloud)
3. Click **New App**
4. Select repo & branch
5. Add environment variable:

```
GROQ_API_KEY=your_key
```

6. Deploy 🚀
